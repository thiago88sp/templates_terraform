## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/thiago88sp/templates_terraform/tree/main/redis

# Access
$ cd redis

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
